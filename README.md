# Enviredit
A free and open source text editor

![Enviredit](https://github.com/user-attachments/assets/9ada4134-1ba6-4e8e-aff1-07dd70b0a29f)

## Requirements
- Dotnet SDK 9.0 installed on your system

## Installation and Usage
Clone the git repository, then cd into the cloned repository.
From there run the following command:
```shell
dotnet run
```
This will build and run the project
